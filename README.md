# GitHub User Finder

**Автор:** [Фукс А.В.]

## Описание
GUI-приложение для поиска пользователей GitHub через API. Позволяет искать по логину, просматривать результаты в таблице, добавлять понравившихся пользователей в избранное и сохранять список избранных в JSON-файл для последующего использования.

## Возможности
- Поиск пользователей GitHub по имени (логину) через публичный API.
- Отображение результатов в таблице: логин и ссылка на аватар.
- Добавление выбранных пользователей в избранное (с проверкой на дубликаты).
- Просмотр и удаление пользователей из списка избранных.
- Автоматическое сохранение избранных в `favorites.json` и загрузка при запуске.
- Проверка ввода: поисковое поле не должно быть пустым.
- Асинхронный поиск (в отдельном потоке), интерфейс не «зависает».

## Требования
- Python 3.6 или выше.
- Только стандартные библиотеки: `tkinter`, `json`, `urllib.request`, `threading`, `os`. Дополнительные модули не требуются.
- Доступ в интернет для выполнения запросов к GitHub API.
